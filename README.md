## ✨ Features
- 🤖 **Groq AI**: Uses `llama-3.3-70b-versatile` for fast & smart replies
- 🗣️ **Hinglish Support**: Natural Hindi + English mix conversations  
- ⚡ **Vercel Edge**: Deployed on Vercel for lightning fast performance
- 📱 **PWA Ready**: Install as app on mobile/desktop
- 🎨 **Modern UI**: Clean React interface with Markdown support
- 🔒 **Serverless**: Secure API key handling via Vercel Functions
